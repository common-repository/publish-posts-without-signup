Plugin to allow guests of your blog create posts without signup

== Description ==

Plugin to allow guests of your blog create posts without signup or sign in. You have some options as: set howmany posts can be created per ID daily, default user and category to create new posts, etc.

== Installation ==

Follow the below steps to intall the plugin:
1- Send plugins's folder to plugins path, it should be /wp-content/plugins/
2- Access the blog panel and activate the plugin
3- Now, in the Settings area, a new link will appear "Pubish Posts Without Signup", click on that link to go to plugin options page
4- Change the options you want and click on button Save Changes

To start showing the form you must use the chartag [publish_posts_without_signup], put it right on the place you want the form appears.
